## 博客结构

本站使用 [Jekyll](https://jekyllrb.com) 搭建，托管在 GitHub Pages 上，目录结构如下：

```
.
├── _blog/       # 博客文章（Markdown 格式）
├── _resource/   # 图片、音频、视频等静态资源
├── _layouts/    # HTML 模板
└── assets/      # CSS、JS 等前端资源
```

`_blog/` 中的每篇文章都是一个 Markdown 文件，Front Matter 中可以指定标题、日期和标签。

引用 `_resource/` 中资源的方式：

```markdown
![示例图片](/resource/example.png)
```
